## Doodle Studio for Unity

In 2018 I released a commercial plugin for drawing inside of the Unity 3D editor. The plugin lets users quickly make frame by frame animations and add them to the game scene, reminiscent of Adobe Flash.

* See: [Doodle Studio 95! for Unity in the Unity Asset Store](https://assetstore.unity.com/packages/tools/painting/doodle-studio-95-108263)
